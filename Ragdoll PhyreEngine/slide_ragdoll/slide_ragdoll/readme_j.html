ラグドール チュートリアル : スライド

概要

本チュートリアルは、ラグドールを坂に沿って転がすための動作を制御するための手法を紹介します。

スクリーンショット


描画結果

詳細

ラグドールと地形との干渉をトリガ剛体によってチェックし、衝突した面（壁）にSlide属性が指定されていた場合に発動します。 ラグドールを坂に沿って転がすような動作を表現します。ラグドールをそのまま坂の上に置くと、たいていのケースではそのまま滑り落ちます。実際にはコリジョン用の地形は平らになっていたとしても、レンダリング用の地形には高精細な凹凸が表現されているケースが多く、単に滑り落ちてくるだけでは視覚的に不自然に感じられます。 そこで、指定した角速度で転がりながら、ラグドールが坂を落ちてくるような動作を与えます。スムーズに転がすため、ラグドールの体幹軸が坂に対して水平になるように同時に姿勢を回転させ、坂の傾斜に応じた、滑り落ちるための並進速度を与えています。 複雑な地形の上にラグドールが乗り上げ、スタックした場合に、Slideによってラグドールを安全なエリアへ転がし、状況を回避する目的に使用することもできます。

操作説明

アクション 挙動
○ button シミュレーションをステップ実行
△ button ラグドールをランダムな方向に投げる
× button シミュレーションを開始/停止
up button カメラを上方に回転
right button カメラを右に回転
left button カメラを左に回転
down button カメラを下方に回転
R2 button シーンのリセット
R1 button カメラを前方に移動
L2 button 剛体をつかむ・はなす
L1 button カメラを後方に移動

タッチパッドを使って、つかんだ剛体を動かすことができます。
